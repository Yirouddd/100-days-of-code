## [241] Different Ways to Add Parenthesses

**Difficulty**: ⭐️⭐️ Medium  
**Date**: 2026-03-01     
**Tags**: `Divide and Conquer`   

### Problem Description  
Given a string expression of numbers and operators, return all possible results from computing all the different possible ways to group numbers and operators. You may return the answer in any order.

The test cases are generated such that the output values fit in a 32-bit integer and the number of different results does not exceed 104.

### 解题思路

#### 1. 核心思想：按运算符分治

- 遍历字符串中的每一个字符  
- 当遇到运算符（`+`、`-`、`*`）时，将表达式**分治**为左右两部分  
- 左半部分和右半部分分别**递归计算**所有可能的结果  

#### 2. 递归计算过程

```cpp  
vector<int> left = diffWaysToCompute(左侧表达式);  
vector<int> right = diffWaysToCompute(右侧表达式);  

#### 3. 边界条件（Base Case）  
如果当前表达式是纯数字（不含任何运算符）  

直接返回 [数字] 作为结果



## [932] Beautiful Array

**Difficulty**: ⭐️⭐️ Medium  
**Date**: 2026-03-01     
**Tags**: `Divide and Conquer`   

### Problem Description
Given the integer n, return any beautiful array nums of length n. There will be at least one valid answer for the given n.

### 解题思路
奇偶分离：把所有奇数放在左边，偶数放在右边  
因为奇数+奇数=偶数，偶数+偶数=偶数，它们的平均值是整数  
但奇数+偶数=奇数，平均值不是整数 → 不可能相等  
  
"奇数放左，偶数放右，左右递归，1变奇偶"  
  
base case  
n=1时 → [1]
  
recursion:  
beautiful(n) =   
  左边 = beautiful( (n+1)/2 ) 的每个数 ×2-1  
  右边 = beautiful( n/2 )     的每个数 ×2  
  左边 + 右边 拼起来
